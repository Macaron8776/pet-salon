# 【ペットサロン　イメージサイト】

ペットサロンの公式ホームページを想定した、イメージサイトです。

## 💡 開発背景と目的

* HTML・CSS勉強のために作成をしました。自宅にペットがおりますため、ペットサロンを題材として作成をしました。

---

## 🚀 主な機能・意識した内容

* ** ヘッダー・フッターの作成 **：メイン画面のヘッダーに【ロゴ画像】・【タブメニューの配置】を行い、レイアウトを整えました。
* ** レイアウトの意識 **：サイト内のレイアウトとして【メインビジュアル】・【初めてのお客様】・【料金・システム】・【代表者からの一言】などのグループに分けてサイトを構成しました。
